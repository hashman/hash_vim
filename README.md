# Hashman Vim

## Requirement
1. vim
2. git
3. curl

```bash
$ apt-get update
$ apt-get install vim git curl
```

## Installation

```bash
$ git clone https://github.com/hashman/hash_vim.git
$ cd hash_vim
$ ./install.sh
```
